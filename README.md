# minecraft-daycounter-coords
A simple yet useful datapack for Minecraft that adds a day counter and a real-time coordinate display to the action bar.  

## ✨ Features Day Counter: 
- Displays the current day in the world in Minecraft
- Live Coordinates: Shows the player's XYZ coordinates updated in real time
- Action Bar Display: Everything displayed in the action bar for quick, non-invasive reading

## 📋 Information 
- Pack Format: 1.20.1+
- Functions: Load and tick (continuously updating) functions
